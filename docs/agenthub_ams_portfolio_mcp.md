# AMS 포트폴리오 라이브 조회 가이드

## 사용 원칙

등록된 인증형 AgentHub 서비스를 통해 AMS 포트폴리오의 최신 상태를 조회할 수 있다.
라이브 조회는 최신 자산·수익률·MDD가 필요한 경우에 사용하며, 내부 연결 설정과
배포 세부 정보는 공개 레포에 기록하지 않는다.

정적 `portfolio_cache.json`은 오프라인 검토·감사·특정 시점 재현을 위한 마지막
스냅샷으로 사용한다. 라이브 경로를 사용할 수 없는 경우에만 정적 캐시로 대체한다.

## 결과 사용 시 유의사항

- 현재 보유 중이며 원화 평가금액이 양수인 항목만 포함한다.
- 원가 계산에 필요한 평균단가 또는 평가 기준값이 없는 항목은 원가·평가손익·수익률이
  `null`일 수 있다.
- 라이브 결과와 정적 캐시의 스키마 변경은 함께 검증한다.

## 운영 경계

연결 주소, 인증 방식, 내부 도구 식별자, 배포와 갱신 절차는 인증된 운영 문서에서만
관리한다. 이 레포에는 비밀값이나 내부 인프라 식별 정보를 추가하지 않는다.
